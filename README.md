# 🛒 GreenCart

A modern full-stack **MERN Stack E-Commerce** application that provides a seamless online shopping experience with secure authentication, product management, shopping cart functionality, and order processing.

🌐 **Live Demo:** https://green-cart-theta-one.vercel.app/

---

## 🚀 Features

### 👤 User Features
- Secure User Authentication
- Browse Products by Categories
- Search Products
- Add / Remove Items from Cart
- Manage Delivery Addresses
- Place Orders
- View Order History
- Responsive Design for Mobile & Desktop

### 🛠️ Admin Features
- Admin Authentication
- Add New Products
- Update Product Details
- Delete Products
- Manage Orders
- Role-Based Access Control

## ⚡ Getting Started

### 1. Clone the Repository

```bash
git clone <your-repository-url>
cd GreenCart
```

### 2. Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** directory.

```env
PORT=4000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

CLIENT_URL=http://localhost:5173
```

For the frontend, create a `.env` file inside the **client** directory.

```env
VITE_BACKEND_URL=http://localhost:4000
```

---

## ▶️ Run Locally

Backend

```bash
npm run server
```

Frontend

```bash
npm run dev
```
